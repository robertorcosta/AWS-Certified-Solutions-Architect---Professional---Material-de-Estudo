# AWS Certified Solutions Architect – Professional (SAP-C02)

Material de estudo em Português do Brasil para a certificação AWS Solutions Architect Professional.

## 📁 Conteúdo

| Arquivo | Descrição |
|---------|-----------|
| `aws-sap-c02-guia.html` | Guia completo com navegação lateral, seções colapsáveis e referências oficiais |
| `aws-sap-c02-mapa.html` | Mapa mental interativo para revisão rápida (bate-pronto) |

## 🚀 Como usar

Abra os arquivos `.html` diretamente no navegador. Não precisa de servidor ou dependências.

- **Guia Completo** → estudo aprofundado, tabelas comparativas, cenários de prova
- **Mapa Mental** → revisão rápida, clique para expandir cada serviço

Os dois HTMLs possuem links cruzados entre si.
